Keyboard Controls

We have implemented several different commands for playback control via keyboard shortcuts:

  • Play/Pause
  • Mute/Unmute
  • Enter Fullscreen
  • Exit Fullscreen
  • Seek + 5 seconds
  • Seek + 10 seconds
  • Seek - 5 seconds
  • Seek - 10 seconds
  • Volume + 5%
  • Volume + 10%
  • Volume - 5%
  • Volume - 10%
  • Space
  • M
  • F
  • Esc
  • right
  • CTRL + right
  • left
  • CTRL + left
  • up
  • CTRL + up
  • down
  • CTRL + down

Our implementation is open for additional keyboard shortcuts.
